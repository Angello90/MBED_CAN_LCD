# Projet: Utilisation d'un écran LCD en CAN avec MBED

## Description
Ce projet implémente une classe permettant de contrôler un écran LCD via le protocole CAN (Controller Area Network) en utilisant la plateforme MBED. L'objectif est de faciliter l'intégration et la gestion des écrans LCD dans des projets embarqués nécessitant une communication CAN.

## Fonctionnalités
- Ecriture sur l'écran LCD en CAN
- Position du curseur (en x et y)
- Effacer l'écran 

## Utilisation
```cpp

#include "lcd_can.h"
LCD_CAN lcd;
int val = 10;
lcd.setCursor(); // x=0, y=0
lcd.print("Valeur: %d", val);
lcd.clear();
lcd.setCursor(1, 1);
lcd.print("Valeur: %d", val);
```
